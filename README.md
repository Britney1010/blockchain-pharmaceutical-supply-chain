# Blockchain-Based Pharmaceutical Supply Chain Management System
Blockchain-Based Pharmaceutical Supply Chain Management System Using Smart Contracts

## Project Overview
This project investigates the use of blockchain technology and smart contracts to enhance transparency, traceability, and regulatory compliance in pharmaceutical supply chains, with a focus on the Kenyan context.

## Research Objectives
- To examine challenges affecting pharmaceutical supply chain integrity
- To evaluate the applicability of blockchain and smart contracts in addressing these challenges
- To propose a blockchain-based framework for pharmaceutical supply chain management

## Technologies (Planned)
- Blockchain (Ethereum / Hyperledger Fabric)
- Smart Contracts (Solidity)
- Backend APIs
- Web-based User Interface

- Chapter 2: In Progress
